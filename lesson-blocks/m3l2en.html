Week 3: Object-Oriented Programming for Aerospace

OOP means organizing code into reusable objects. Instead of writing scattered functions, you define classes that bundle data and behavior together. In aerospace, this is how we think about systems: A spacecraft is an object with properties (mass, altitude, velocity, fuel) and behaviors (burn engines, communicate home, adjust attitude). A rocket is an object with thrust, mass, fuel, and the ability to calculate acceleration and delta-v.

The Rocket Class: A Working Example

class Rocket:
    """Represents a launch vehicle with thrust, mass, fuel capacity, and delta-v calculations."""

    def __init__(self, name, mass_kg, thrust_n, isp_seconds):
        """Initialize a rocket.

        Args:
            name: Rocket designation (e.g., "Falcon 9", "New Shepard")
            mass_kg: Dry mass (structure, engines, avionics) in kg
            thrust_n: Sea-level thrust in Newtons
            isp_seconds: Specific impulse in seconds (ISP is exhaust efficiency)
        """
        self.name = name
        self.mass_kg = mass_kg
        self.thrust_n = thrust_n
        self.isp_seconds = isp_seconds
        self.fuel_kg = 0
        self.altitude_m = 0
        self.velocity_mps = 0

    def add_fuel(self, fuel_kg):
        """Load fuel into the rocket."""
        self.fuel_kg = fuel_kg
        print(f"{self.name}: Loaded {fuel_kg} kg of propellant")

    def total_mass(self):
        """Total mass including fuel."""
        return self.mass_kg + self.fuel_kg

    def calculate_acceleration(self):
        """Calculate current acceleration using F=ma."""
        G = 9.81  # Earth surface gravity in m/s^2
        thrust_eff = self.thrust_n - (self.total_mass() * G)  # Subtract weight
        return thrust_eff / self.total_mass()

    def calculate_delta_v(self):
        """Calculate total delta-v available (exhaust velocity times ln(mass ratio)).

        Delta-v is the total velocity change the rocket can achieve from all fuel burns.
        Formula: dv = isp * g * ln(initial_mass / final_mass)
        where g = 9.81 m/s^2 (standard gravity)
        """
        import math
        G = 9.81
        exhaust_velocity = self.isp_seconds * G
        initial_mass = self.total_mass()
        final_mass = self.mass_kg  # After all fuel is burned

        if final_mass <= 0:
            return 0

        mass_ratio = initial_mass / final_mass
        delta_v = exhaust_velocity * math.log(mass_ratio)
        return delta_v

    def status(self):
        """Print rocket status."""
        print(f"\n{self.name} Status:")
        print(f"  Dry mass: {self.mass_kg} kg")
        print(f"  Fuel: {self.fuel_kg} kg")
        print(f"  Total mass: {self.total_mass()} kg")
        print(f"  Thrust: {self.thrust_n:,} N")
        print(f"  ISP: {self.isp_seconds} seconds")
        print(f"  Delta-v available: {self.calculate_delta_v():.0f} m/s")
        print(f"  Current acceleration: {self.calculate_acceleration():.2f} m/s²")

# Create a Falcon 9 first stage
falcon9 = Rocket("Falcon 9 F1", mass_kg=25_000, thrust_n=5_100_000, isp_seconds=282)
falcon9.add_fuel(385_000)
falcon9.status()

# Create New Shepard (suborbital)
shepard = Rocket("New Shepard", mass_kg=6_000, thrust_n=110_000, isp_seconds=210)
shepard.add_fuel(24_000)
shepard.status()

Week 4: NumPy and Pandas for Aerospace Data

NumPy provides fast arrays and mathematical operations. Pandas provides tabular data (DataFrames) which is how real telemetry is organized. Every aerospace engineer uses these libraries to crunch mission data.

Example: ISS Altitude Data Analysis with Pandas

import pandas as pd
import numpy as np

# Sample ISS altitude data over time (real format from space agencies)
data = {
    'timestamp': ['2026-04-22 08:00', '2026-04-22 08:05', '2026-04-22 08:10', '2026-04-22 08:15'],
    'altitude_km': [408.2, 408.1, 407.9, 407.8],
    'velocity_mps': [7659, 7658, 7660, 7659],
    'temperature_c': [22.1, 22.0, 22.3, 22.1]
}

df = pd.DataFrame(data)
print(df)
print(f"\nAverage altitude: {df['altitude_km'].mean():.2f} km")
print(f"Min altitude: {df['altitude_km'].min():.2f} km")
print(f"Max velocity: {df['velocity_mps'].max()} m/s")

# Detect anomalies (altitude drops more than 0.2 km)
df['altitude_change'] = df['altitude_km'].diff()
anomalies = df[df['altitude_change'] < -0.2]
if len(anomalies) > 0:
    print("\nAnomalies detected (rapid altitude loss):")
    print(anomalies)

Week 5: REST APIs and Live Data

APIs (Application Programming Interfaces) let your code ask servers for data. NASA provides free APIs. SpaceX provides an API. N2YO provides real-time satellite tracking. Use the requests library to fetch data in JSON format, then parse it with Python.

API Organization Data Provided Python Example
NASA API NASA APOD (Astronomy Photo of Day), Mars rovers, exoplanets requests.get("api.nasa.gov/...?api_key=...")
SpaceX API SpaceX (community maintained) Launch history, rocket specs, missions requests.get("api.spacexdata.com/v4/launches")
N2YO Satellite Tracker N2YO Real-time TLE, satellite passes, positions requests.get("api.n2yo.com/rest/v1/satellite/...")
Open-Meteo Open-Meteo (no key needed) Weather, atmospheric data requests.get("api.open-meteo.com/v1/forecast")
USGS Earthquake USGS Seismic activity (used to study Earth from orbit) requests.get("earthquake.usgs.gov/earthquakes/feed/v1.0/...")

Example: Query SpaceX Launch History

import requests
import json

# Query SpaceX API for recent launches (no auth required)
url = "https://api.spacexdata.com/v4/launches?limit=5&sort=-date_unix"
response = requests.get(url)
launches = response.json()

for launch in launches:
    name = launch.get('name', 'Unknown')
    date = launch.get('date_utc', 'Unknown')
    success = launch.get('success', None)
    status = "Success" if success else "Failure" if success is False else "Pending"
    print(f"{name} - {date[:10]} - {status}")

Week 6: Data Visualization with Matplotlib

Raw data is hard to understand. Plots tell the story. Plot orbital trajectories, satellite passes, launch windows, telemetry trends.

import matplotlib.pyplot as plt
import numpy as np

# Plot an orbital trajectory
time = np.linspace(0, 90, 360)  # 90 minutes, every 15 seconds
altitude = 408 + 2 * np.sin(time / 90 * np.pi)  # Simulated altitude variation
velocity = 7660 - 10 * np.cos(time / 90 * np.pi)  # Simulated velocity

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 6))

ax1.plot(time, altitude, 'b-', linewidth=2)
ax1.set_ylabel('Altitude (km)', fontsize=11)
ax1.set_title('ISS Orbital Profile (1 Orbit)', fontsize=13)
ax1.grid(True, alpha=0.3)

ax2.plot(time, velocity, 'r-', linewidth=2)
ax2.set_xlabel('Time (minutes)', fontsize=11)
ax2.set_ylabel('Velocity (m/s)', fontsize=11)
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

Answer Key and Explanations

Question 1 (Satellite class): Class must have __init__ with four properties. Method must calculate mean motion correctly. Test with realistic values (ISS: altitude 408 km, period 90 minutes, should yield ~16 revolutions/day).

Question 2 (SpaceX API): Must use requests.get() with correct SpaceX API v4 endpoint. Parse JSON response correctly. Extract launch name, date_utc, success fields. Handle missing fields gracefully.

Question 3 (Pandas filtering): DataFrame must have correct column types. Filter using df[df['success'] == True]. Success rate = (count of True) / (total rows) * 100.

Question 4 (Matplotlib): Must create two y-axes using twinx(). Plot lines on each. Include legends, axis labels, title. Both lines visible without overlap.

Question 5 (NumPy orbital velocity): Formula v = 2*pi*R / T. R = Earth radius + altitude. T = orbital_period_minutes * 60. ISS example: v ~ 7660 m/s.

Question 6 (OOP explanation): Classes bundle data and behavior. Rocket class encapsulates mass, thrust, fuel, and behaviors like calculate_delta_v(). Makes code reusable and maintainable. Mirrors physical reality.

Question 7 (Mini-project design): Must identify a specific data source (API), analysis question, and visualization type.
